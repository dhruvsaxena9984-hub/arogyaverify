# ArogyaVerify

ArogyaVerify is a health verification application built with HTML, CSS, and JavaScript.

## Deployment

This project is configured for automatic deployment on Netlify.

### Deploy to Netlify

Click the button below to deploy:

[![Deploy to Netlify](https://www.netlifybutton.com/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/dhruvsaxena9984-hub/arogyaverify)

### Manual Deployment

1. Go to [Netlify](https://www.netlify.com)
2. Connect your GitHub repository
3. Set Build command to: (leave empty)
4. Set Publish directory to: `.`
5. Deploy

## Local Development

Simply open `arogyaverify.html` in your web browser.
